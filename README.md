# Парсер сайта pedsovet.org

Простой парсер для сайта pedsovet.org, написанный на Python с использованием BeautifulSoup.

## Описание проекта

Проект демонстрирует основы парсинга веб-страниц:
- Получение HTML-страниц с помощью requests
- Анализ и извлечение данных с BeautifulSoup
- Сохранение результатов в JSON
