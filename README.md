# Парсер статей с Pedsovet.org

## Описание проекта

Этот проект представляет собой парсер для извлечения информации о педагогических статьях с сайта pedsovet.org. Программа автоматически находит карточки статей на главной странице, извлекает заголовки и ссылки на полные тексты.

## Извлекаемые данные

Для каждой статьи программа извлекает:
- **Заголовок** (наименование статьи)
- **Ссылку** на полную версию статьи

## Источник данных

**HTML берется непосредственно с сайта**: https://pedsovet.org/

**Подготовка HTML не требуется** - программа сама:
- Загружает страницу через интернет
- Обрабатывает HTML структуру
- Извлекает нужные данные

## Технологии

- **Python 3.7+**
- **BeautifulSoup4** - парсинг HTML
- **Requests** - HTTP-запросы
- **lxml** - XML-парсер (бэкенд для BeautifulSoup)

## Установка зависимостей

Перед запуском необходимо установить требуемые библиотеки:

```bash
pip install -r requirements.txt
```

## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone https://github.com/shaurmafia/parser.git
cd parser
```
### 2. Создание виртуального окружения
```
python -m venv venv
venv\Scripts\activate
```
### 3. Установка зависимостей
```
pip install -r requirements.txt
```
### Запуск программы
```
python parse_pedagogy.py
```
