# Парсер карточек статей с сайта pedsovet.org

## Описание проекта

Программа представляет собой парсер веб-страниц, который извлекает информацию о статьях с главной страницы сайта pedsovet.org. Парсер находит карточки статей, извлекает их заголовки и ссылки на полные версии статей.

## Что делает программа

Программа выполняет следующие действия:
1. Загружает главную страницу сайта pedsovet.org
2. Находит на странице карточки статей (прямоугольные виджеты с материалами)
3. Для каждой карточки извлекает:
   - Наименование статьи (заголовок/подпись на карточке)
   - Ссылку на полную статью
4. Сохраняет результаты в структуру Python (список словарей)
5. Выводит результаты в консоль и сохраняет в JSON-файл

## Источник HTML и подготовка
HTML-код получается напрямую с сайта https://pedsovet.org/. Программа автоматически:

Выполняет HTTP-запрос к сайту

Получает актуальную версию главной страницы

Преобразует HTML в объект BeautifulSoup для дальнейшего анализа

Никакой дополнительной подготовки HTML не требуется - программа работает с "живой" версией сайта.


## Просмотр результатов
После выполнения программы результаты можно увидеть:

В консоли - программа выводит список найденных статей с их заголовками и ссылками

В файле articles.json - все данные сохраняются в структурированном JSON-формате


## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone https://github.com/shaurmafia/Parser.git
cd Parser
```
### Создание виртуального окружения
```
python -m venv venv
venv\Scripts\activate
```
### Установка зависимостей
```
pip install -r requirements.txt
```
### Запуск программы
```
python parser_ped.py
```
